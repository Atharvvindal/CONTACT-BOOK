# 📒 Contact Book (Python)

A beginner-friendly Contact Book application built with Python. The program allows users to store and manage contacts in a JSON file, making the data persistent even after closing the application.

---

## ✨ Features

- ➕ Add new contacts
- 🔍 Search contacts by name
- ✏ Edit existing contacts
- ❌ Delete contacts
- 📋 View all saved contacts
- 💾 Stores contacts in a JSON file
- 🚫 Prevents duplicate contact names
- 🔄 Interactive command-line menu

---

## 📂 Technologies Used

- Python 3
- JSON

---

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/YOUR_USERNAME/Contact-Book-Python.git
```

2. Navigate to the project folder

```bash
cd Contact-Book-Python
```

3. Run the program

```bash
python contact_book.py
```
